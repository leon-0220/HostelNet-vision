HostelNet Vision
This is a web-based hostel management system development as part of a student project. The system includes features such as user registration, login, room booking, check-in, and check-out functions.

Note (In Progress)
This project is still under development. Currently, there are known issues with the following modules:
- user registration
- user login
- room booking
- check-in & check-out function
- user dashboard

These errors are being actively worked on and will be fixed in upcoming updates. The code is available for reference and review.

Features
- admin & user dashboards
- room listing and booking system
- check-in / check-out management
- registration & login system

Tech stack
- HTML
- JavaScript
- CSS
- MySQL
